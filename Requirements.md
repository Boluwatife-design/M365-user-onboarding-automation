
---

## 🔧 Requirements

- Microsoft 365 Tenant  
- Power Automate  
- Microsoft Entra ID  
- Group-Based Licensing configured  
- (Optional) Azure AD P1/P2 for Dynamic Groups  

---

## 📥 Setup Guide

1. Import the flow from `/flow/onboarding-flow.zip`
2. Update connections (Entra ID, SharePoint, etc.)
3. Configure your data source (SharePoint or HR system)
4. Map user attributes correctly
5. Ensure groups already have licenses assigned
6. Test with a sample user

---

## 🚀 Optional Enhancements

- Use **Dynamic Groups** for auto-assignment  
- Add approval flow (HR → IT)  
- Send welcome email to new users  
- Integrate with Teams notifications  
- Add logging (SharePoint / Excel / Log Analytics)
